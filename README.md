# class-notify
# 🔔 딩동, 교실 (Ding-Dong Classroom)

**"행복한 우리 교실을 실시간으로 긴밀하게 연결합니다"**

정보교사로서 교실 환경을 개선하고, 학생들과의 소통을 효율화하기 위해 개발한 '스마트 학급 알림 및 호출 시스템'입니다.

---

## 🚀 서비스 이용 안내
실제 운영 환경을 확인하시려면 아래 링크를 클릭해 주세요.

👉 **[지금 바로 학급 알림판 실행하기](https://shyeonq15.github.io/class-notify/)**

---

## 💡 프로젝트 소개
기존의 아날로그 방식이나 교내 메신저만으로는 부족했던 실시간 학급 전달사항을 효과적으로 관리하기 위해 제작했습니다. 구글 앱스 스크립트(GAS)를 활용하여 별도의 서버 구축 없이도 저비용, 고효율로 운영 가능한 시스템입니다.

## 🛠 주요 기능
*   **교사용 대시보드:**
    *   조종례 및 교과 전달사항 다중 학급 일괄 발송
    *   마감 기한 설정 및 실시간 연장/종료 제어
    *   학생 호출 명단 선택 및 실시간 호출
*   **학생용 알림판:**
    *   교실 내 TV 모니터와 연동하여 실시간 공지 확인
    *   긴급 호출 시 즉시 알림창 활성화

---

## 🔒 개인정보 보호 및 데모 안내 (중요)
본 프로젝트는 교육 현장에서의 학생 개인정보 보호(Privacy by Design)를 최우선 가치로 삼고 설계되었습니다.

*   **더미 데이터 활용:** 평가 및 시연을 위해 `StudentList` 시트에는 실제 학생 정보가 아닌 **1-1반의 더미 데이터**만 포함되어 있습니다. 실제 학교 현장에서 운영 시에는 적절한 개인정보 암호화 및 비식별화 처리를 필수적으로 수행해야 합니다.
*   **인증 코드:** 데모용 학교 인증 코드는 `2026학교호출`로 설정되어 있습니다.
